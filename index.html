Documentação do Projeto: Nome do Projeto

Alien Maluco

Motivação: Pois promove o aprendizado de lógica de programação de maneira lúdica, utilizando uma abordagem prática e interativa.

Objetivo: Ajudar a implementar a lógica de programação no ensino infantil de uma forma dinâmica e interativa.

Contextualização: O dispositivo é um jogo feito para ser jogado com botões de arduino, servindo como diversão, entretenimento e de material educativo.

Componentes e Suprimentos

  • Placa de Arduino Uno - Código de referência: A000066
  • Cabo USB compatível com Arduino - Código de referência: UCB-123
  • Computador com Aplicativo Pictoblox - Código de referência: N/A
  • Protoboard - Código de referência: PB400
  • Resistor 10 KO - Código de referência: R10K
  • Botão tipo push - Código de referência: BP-01
  • Jumpers macho-macho - Código de referência: JMM-05

Aplicativos e Plataforma

  • Pictoblox - Plataforma para programação gráfica baseada em Scratch, com suporte para interação com Arduino e robótica. Código de referência: PBX-001
  • Scratch - Aplicativo de programação gráfica voltado para iniciantes, permitindo a criação de animações, jogos e simulações. Código de referência: SCR-002
  • Pictoblox Desktop - Versão para computador do Pictoblox, disponível para Windows e macOS. Código de referência: PBX-DT-003
  • Pictoblox Web - Plataforma online para programação gráfica com funcionalidades de IA e IoT. Código de referência: PBX-WEB-004
  • Scratch Files - Arquivos com extensão .sb3 usados para armazenar projetos de programação no Scratch e Pictoblox. Código de referência: SCRF-005
